Conducted Order and Returns Analytics for a Direct-to-Consumer (D2C) fashion brand using Microsoft Excel, where I cleaned and structured raw order data, validated datasets, and analyzed sales performance, return rates, and customer purchasing behavior to uncover actionable business insights. Leveraged Pivot Tables, Pivot Charts, and advanced Excel functions such as SUMIFS, COUNTIF, IF, and VLOOKUP/XLOOKUP to calculate key performance indicators including total revenue, total orders, return percentage, category-wise and region-wise performance, and revenue loss due to returns. Designed an interactive dashboard to visualize trends and identify high-return product categories, seasonal patterns, and operational gaps, ultimately providing data-driven recommendations to improve inventory planning, reduce return rates, and enhance overall profitability.
